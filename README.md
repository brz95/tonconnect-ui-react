## tonconnect/ui-react
### Стэк: NextJS, tonconnect/ui-react, react-query, react-use, lottie, tailwindcss

### Порядок команд для запуска
```sh
git clone https://github.com/brz95/tonconnect-ui-react.git
```

```sh
cd tonconnect-ui-react
```

```sh
yarn
```

```sh
yarn dev
```
